#Welcome to yjinglee.office#
----------
## 简介 ##

yjinglee.office用于.Net平台下的Excel操作，主要封装NPOI对外提供更简单实用的API，提供以下功能点：

- 读取Excel数据转换成对象集合
- 写入集合到Excel，并提供可以Excel样式定义
- 公开底层Api
